Зачем нужно цифровое портфолио?

Наличие цифрового портфолио студента/специалиста - важный элемент в условиях цифровой экономики;

Работодателей интересуют подтвержденные результаты освоения различных навыков;

Удобство размещения ссылки/QR – кода на цифровых платформах по трудоустройству.

Группировка по типам навыков

В настоящее время выделяют 3 основных навыка:

Hard - Skills (профессиональные навыки);

Soft - Skills (надпрофессиональные навыки);

Self - Skills (личностные навыки);
